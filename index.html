FAQ



How do I play?
Movement: You can either hold W, A, S, D to manually move in that direction, press the buttons with 1 arrow to move 1 block in that direction, press any of the arrow keys to automine in that direction, or press the buttons with 3 arrows to automine in that direction.
Rare Spawns One thing you have to pay attention to is spawn sounds. Rare ores all have a noise to indicate they have been spawned in the mine, and it is generally recommended to always collect these ores.
Crafting Crafting is the most important part of the game. It allows you to create better pickaxes and make gears that will help you progress. New pickaxes will either let you mine more blocks through abilities or gain more luck. Gears are just like upgrades in other games and can increase luck, speed, and other silly things that will help with progression.

The Menu The menu contains a lot of important tabs that you should make note of!
Settings The settings tab allows you to change many things about the game (explained in the settings section). It is also where you can control spawn sound volume, and determine what spawn sounds get played.
Data The data tab has everything with your data. Pressing export will paste your save file into the text box, and asks if you want to download it as a file. It is recommended to make backups of your data regularly in case browser data is lost. Paste data into the text box and press import to import that data. This will overwrite your current data, so make sure it is what you want. Importing nothing performs a hard reset on your game.
Logs The logs tab is where you can generate ore logs for your current session. This only logs Flawless tier and above ores, along with any rare cave ores. These are also how you can provide proof of ores in the Discord.
Statistics The statistics tab shows you some things about yourself, such as total ores owned in the rare tiers, total time played, session time, blocks mined, and caves generated (more statistics will come eventually).
Ore Index The ore index tab gives you information on the ores in each layer in the world you are currently in. Higher tiers will be blacked out until the ore is found. Each index card will also tell you the rarity of the ore, how rare it is with your current luck, and what variants of the ore you own.
Locations The locations tab provides some very helpful areas. The first one is the forge, which allows you to craft certain ores you might need, or fission rare ores you have into a bunch of more common ores. The second is variant conversion, which can be more easily accessed by clicking on an ore in your inventory that is a variant. It will allow you to convert these variants into their normal selves, gaining more the rarer the variant. The third location is the workshop, which is currently only visible in Subrealm 1. The workshop will allow you to upgrade certain items, making them work better than usual.

Settings
Simulated RNG This setting changes block generation to calculate averages of what you should get in certain amounts of blocks generated opposed to rolling for each ore individually. Heavily increased game performance, slightly lowered RNG accuracy. Any generation above 35,899 blocks will have a different RNG formula applied to it.
Minimum Speed This setting allows you to set a cap on your speed in case the game runs slowly. It will slow you down if you exceed this speed.
Capacity This setting allows you to change your mine capacity. It is recommended to keep it under 5,000,000 as there can be performance issues with high mine capacities.
Latest Ores Max Length This setting allows you to change the amount of ores that will show up in latest spawns and latest finds.
Block Update Speed This setting allows you to change the speed at which the display will update when automining. (uses ms)
Stop On Rare This setting is one of the more important ones, and will stop you mining if you spawn an ore of the specified tier. Click on tiers and select which tiers you would like to be automatically stopped on.
Minimum Spawn Message This setting sets the tiers an ore has to be for it to generate a spawn message and appear in latest spawns and latest finds.
Travel To Index On Click These settings allow you to specify if clicking an ore in a crafting recipe or in the normal inventory will open the index page for it.
Toggle Block Updates This settings either enables or disables the blocks on the screen, and can increase performance.
Use Numbers For Pickaxe Names This setting switches pickaxe names to become numbers, like "Pickaxe 1" instead of the pickaxes actual name.
Use Different Emojis This setting attempts to switch the emoji font to one that has correct emoji sizes if the current emoji font breaks the display. (doesn't work on all devices)
Invisible Path Blocks This setting switches the white circles to nothing, can negatively impact performance but looks better.
Toggle Caves This setting allows you to switch caves on or off. Caves spawn after Geode Staff and can provide a lot more resources.
1/250m+ RNG Logs This setting changes logs to only include ores that have an RNG of over 1/250,000,000. RNG is how rare an ore is including player luck instead of how rare the ore normally is.
Spawn Effects This setting allows you to turn spawn effects on or off (the shake and color changes).
Automine Protection This setting makes abilities and Infinity Collector unable to mine ores, and you have to manually mine them.
Nyerd This setting changes the ore tracker arrow to the :nyerd: emoji.
AFK Mode Removes most elements on the screen to highly increase performance.

Ore Tiers

Antique : 1/750,000+ (750k) Mystical : 1/5,000,000+ (5m) Divine : 1/25,000,000+ (25m) Flawless : 1/160,000,000+ (160m) and the first tier to log normally. Interstellar : 1/750,000,000+ (750m) and the first tier to have spawn effects Metaversal : 1/1,500,000,000+ (1.5b) Sacred : 1/5,000,000,000+ (5b) Ethereal : 1/100,000,000,000+ (100b) Imaginary : 1/1,000,000,000,000+ (1t) Celestial : Secret ores with special spawn conditions, ranges from 1/10,000 (10k) to 1/13,000,000,000,000,000 (13qd)

Caves Caves generate in the mine at a rate of 1/500 after crafting Geode Staff. They will mine a little over 1,500 blocks and will give you whatever was mined. They don't increase blocks mined.
Cave Types Cave types have a chance of generating when a cave is generated. These have special blocks instead of the normal layer, and have their own loot tables. These loot tables contain cave exclusives, which can only be generated in that specific cave type and are not affected by any luck as of now. Any cave exclusive will log, no matter the rarity.
Abysstone Caves Abysstone caves are unlocked with a special, secret gear in World 2. They are the rarest cave type and generate any ore from World 1 and World 2, excluding celestials and any cave exclusives. It has its own exclusive ore, which is celestial tier.

Powerups And Other Things Like Them
Terrestrial Terror Mines a 101x101 block area around the player using the players current luck. The size gets multiplied by log10(your blocks mined / 500,000)
The Spelunker Increases cave size, doubles cave spawn rate, and gives 2x cave luck for 2:30 minutes.
Illogical Randomiser Makes a random ore from the layer you are in 1.5x more common. Will display which ore is chosen.
Re-Repurposed Replicator Makes any common ore that wouldn't usually be affected by luck have up to 3.5x luck applied to it.
Paradoxical Progression Gives you any unowned pickaxe or gear for at least 1 minute. Will not give World 1 pickaxes in World 2, and is disabled in Subworld 1. The given item is wiped on refresh.
Ore Tracker Unlocked by crafting the ore tracker gear. Points you in the direction of the nearest rare ore that had a spawn message and gives you its coordinates upon clicking "Track Ores!". Will also display the variant of the ore if it has one.

Other Questions What are variants? Variants are versions of ores that have a much lower chance to spawn. They have no functional use, but can be converted to normal, giving more ores the rarer the variant. They look special on the display, glowing their respective color. What happens when I dig down? In World 2 and Subrealm 1, the last layer of the world will keep generating infinitely as you go down. In World 1, there's a special mechanic that keeps generating layers the further down you go. These layers are usually repeated versions of the main layers, with a small chance to have something added to them. There are also a few secret layers that have a very low chance of generating. These special layers, along with the lowest point you've been, will be added to the teleport list on generation. An index page for the secret layers will also be created once you have mined a certain amount in those layers.
What is "Void Prevention"? Void prevention is when the game automatically mines a rare ore you spawned because it was near the end of the mine lifespan. It is there to prevent ore loss from the mine resetting.
